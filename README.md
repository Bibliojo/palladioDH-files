# palladioDH-files
Day of DH workshop for Palladio. Data xl and csv files with which to practice.
network graph walkthrough
Palladio treats the first table as the core dataset, and every additional table you add is “extended data.”

So the mental model your students should have is:

Main Table
   +
Extended Table(s)
   =
One connected dataset. The extended tables add information to the rows of the main table, rather than standing alone.
